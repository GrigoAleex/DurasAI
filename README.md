## Vision

A local-first AI assistant that actually helps.  
Think Siri, but better responses on trivia questions. Think Claude coworker, but free and running entirely on your machine.

No cloud dependency. No subscriptions. No hidden compute elsewhere.  

## Non SLOP

### Zero tolerance for AI slop
Using AI is fine. Blindly shipping its output is not.  
Every contributor is responsible for what they submit. If you don’t understand it, don’t commit it.

### Zero tolerance for overengineering
Accessibility > cleverness.

If your solution needs “just a bit more RAM” or “a slightly better CPU”, you already failed.   
Performance problems are architecture problems.

Target baseline:
- CPU: Apple M1 / Intel Core i5-10400 / AMD Ryzen 5 3600 or equivalent
- ~8GB RAM
- Real-time interaction (no noticeable lag)

If it doesn’t run well there, it doesn’t belong here.
